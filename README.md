# HTML5 File Storage Layout

## Description
This project is a basic HTML5 layout recreation inspired by the general structure and purpose of **MediaFire** (a file hosting and sharing website). This is an educational assignment focused on semantic HTML5 structure.

## Important Note
This is **NOT** a copy of MediaFire. All content, text, and structure are original and created independently for learning purposes. The inspiration comes from understanding the general concept of a file storage/sharing service, but no code, design, or content was copied from MediaFire or any other website.

## Features
- Semantic HTML5 structure
- Mobile-responsive design (readable at 320px width)
- Accessible with skip-link navigation
- Clear, blocky layout with distinct sections
- No JavaScript required
- Minimal inline CSS for structural layout only

## Assignment Criteria Met
✅ `lang` attribute on `<html>`  
✅ One `<h1>` on each page  
✅ `<header>` contains brand and `<nav>`  
✅ `<main>` exists  
✅ Content divided into `<section>`/`<article>` blocks  
✅ `<footer>` exists  
✅ Skip-link: `<a href="#main">Skip to content</a>`  
✅ Passes W3C validator without errors  
✅ Readable at 320px width  
✅ Repository name is 'html5'

## Technologies Used
- HTML5
- Minimal inline CSS (for borders and flexbox layout only)
